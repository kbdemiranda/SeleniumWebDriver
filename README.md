# Selenium WebDriver

<p>Projeto Selenium, contendo alguns testes e tarefas que foram automatizadas utilizando o WebDriver como ferramenta</p>

# Leia-me :) 

<h2>Pré-requisitos</h2>
<ul>
	<li>Possuir o JDK e o JRE instalado</li>
	<li>Sistema Operacional (Windows, Linux e MacOS) com arquitetura de 64 Bits</li>
</ul>

<p>Os WebDriver desse projeto foram selecionados apenas para serem executados em maquinas com arquitetura de 64 Bits, caso deseje executar esse projeto em uma máquina com arquitetura de 32 Bits, substitua os drivers que se localizam dentro da pasta Webdrivers na raiz do projeto pelas versões adequadas </p>
